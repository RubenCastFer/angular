Mi proyecto de Angular

Por Rubén Castellano Fernández

Mi aplicación está basada en la api de de chiste aleatorios de Chuck Norris.

Mediante dos servicios voy agregando las frases a una tabla en donde el usuario puede verlas.

Con el primer servicio se recoge todas las categorías y mediante un select el usuario puede seleccionar la categoría.

En el segundo servicio llamo a de nuevo a la api añadiendo la categoría que el usuario ha seleccionado de modo que la api devolverá una frase relacionada con la categoría agregándola a la tabla, y dándole la posibilidad de borrarla.
